### Code Style

- Add tests for new functionality
- Ensure all tests pass: `pnpm test`
- Maintain or improve code coverage

### Before Submitting

```bash
# Format code
pnpm run format

# Lint code
pnpm run lint

# Type check
pnpm run type-check

# Build
pnpm run build

# Test
pnpm run test
```
